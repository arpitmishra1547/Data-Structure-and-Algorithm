5️⃣ Implement strStr() / Index of First Occurrence (substring search)

Question:
Implement a function int strStr(String haystack, String needle) that returns the index of the first occurrence of needle in haystack.
Return -1 if needle is not part of haystack.
Return 0 if needle is empty.

Input: haystack = "hello", needle = "ll" → Output: 2

Input: haystack = "aaaaa", needle = "bba" → Output: -1

First, solve with a simple brute force:

Outer loop → starting index

Inner loop → compare characters

Then, if you want, later you can explore efficient algorithms (KMP etc.).

Concepts revised:

substring(beginIndex, endIndex)

equals()

Edge cases: empty string, needle longer than haystack

Nested loops with break and continue
